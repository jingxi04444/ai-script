# AI 智能体：爆款短视频脚本生成与复刻平台 - 后台（管理端）需求文档

## 文档信息



* **版本**：V1.0

* **状态**：脚本阶段 / MVP 至壁垒构建期

* **核心目标**：支撑前台用户全流程操作，实现数据管理、合规管控、权限隔离、系统配置，构建产品核心壁垒

* **使用人群**：系统超级管理员、品牌专属管理员、审核员、技术运维人员

## 一、核心原则与核心壁垒

### 核心原则



1. 多租户隔离：确保不同品牌客户的数据独立存储，互不可见，保障数据安全

2. 合规可控：通过违禁词库、审核流程、操作日志，满足合规审计要求

3. 高效支撑：优化 API 调用、数据存储、任务处理流程，保障前台用户使用体验

4. 可扩展性：预留接口与配置项，支持后续接入更多 AI 模型、平台渠道、功能模块

### 核心壁垒



* 私有知识库：沉淀各品牌的 “结构公式”“卖点库”“素材标签体系”，只存特征不存原文，兼顾复用性与合规性

* 全流程管控：从数据采集、脚本生成到素材管理、数据追踪，实现端到端的后台支撑

## 二、功能模块详细说明

### 模块 1：数据采集与解析管理（支撑前台步骤 3）

#### 功能描述

管理视频链接解析相关的 API、插件与日志，确保前台爆款复刻功能稳定运行。

#### 核心功能



1. 解析 API 配置：

* 支持配置多个第三方视频解析服务商（抖音、小红书）的 API Key

* 可设置 API 优先级（主备切换逻辑），当主 API 故障时自动切换至备用 API

* 配置 API 调用频率限制、超时时间、重试次数

* 支持测试 API 连通性，查看当前可用状态

1. 浏览器插件管理：

* 生成 / 管理 Chrome 浏览器插件安装包（供核心客户手动提取视频素材）

* 控制插件版本更新（发布新版本、强制更新、灰度发布）

* 管理插件授权名单（仅授权品牌客户可使用）

1. 解析日志监控：

* 记录每次前台用户的链接解析请求（请求时间、用户 ID、品牌 ID、链接地址、解析结果、耗时、使用 API）

* 支持按时间、品牌、解析状态（成功 / 失败）筛选日志

* 失败日志自动标记原因（如 API 故障、链接无效、权限不足），支持手动重试

1. 数据清洗规则：

* 配置解析数据的清洗逻辑（如去除冗余文本、提取核心文案、标准化数据格式）

* 定义结构化数据提取字段（确保点赞、评论等数据准确映射）

#### 操作权限

仅系统超级管理员、技术运维人员可操作

### 模块 3：生成与审核工作流管理（支撑前台步骤 4）

#### 功能描述

管理脚本审核流程、审核人员、审核记录，确保生成的脚本合规可用。

#### 核心功能



1. 审核席位管理：

* 为每个品牌分配审核员账号（支持多级审核：运营审核→法务审核→经理审核）

* 配置审核员权限（仅可审核本品牌脚本）

* 查看审核员工作负载（待审核数量、已审核数量、审核耗时）

1. 审核任务管理：

* 待审核列表：展示所有品牌待审核的脚本（按提交时间排序）

* 筛选功能：按品牌、脚本状态、提交人、审核阶段筛选

* 任务分配：系统自动分配或手动分配审核任务给审核员

* 催审功能：针对超时未审核的任务，可发送提醒给审核员

1. 审核规则配置：

* 配置合规检查触发条件（如违禁词高亮级别、原创度阈值）

* 配置审核流程（单级审核 / 多级审核，多级审核可设置审核顺序）

* 配置审核通过 / 驳回的通知方式（系统消息 / 邮件 / 短信）

1. 审核记录管理：

* 记录每个脚本的审核轨迹（提交时间、审核员、审核意见、修改记录、通过 / 驳回时间）

* 支持按脚本 ID、品牌、审核时间检索记录

* 记录不可删除，满足合规审计要求

#### 操作权限



* 审核员：仅可查看分配给自己的审核任务，执行审核操作

* 品牌专属管理员：可查看本品牌所有审核任务、审核记录，分配审核任务

* 系统超级管理员：可查看所有品牌审核任务、配置审核规则、管理审核员账号

### 模块 4：视频素材与项目库管理（支撑前台步骤 6-8）

#### 功能描述

管理所有品牌的项目数据与视频素材，确保素材安全、可追溯、可复用。

#### 核心功能



1. 项目管理：

* 每个前台脚本生成任务对应一个后台项目，记录项目状态（草稿 / 审核中 / 生成中 / 已完成 / 已归档）

* 展示项目核心信息（品牌 ID、创建人、创建时间、脚本名称、关联产品、素材数量）

* 支持按品牌、项目状态、创建时间筛选项目

* 支持项目归档、删除（仅超级管理员可删除，删除前需确认）

1. 素材管理：

* 按品牌隔离存储所有素材（视频片段、配音、场景图、参考素材）

* 素材详情页：展示素材关联项目、生成时间、标签、使用次数、存储路径

* 素材操作：支持预览、下载、批量导出、删除（品牌管理员可删除本品牌素材）

* 存储管理：查看各品牌素材存储占用情况，设置存储上限（超出上限提醒品牌清理或扩容）

1. 素材复用统计：

* 统计各素材的复用次数、复用项目、复用效果（关联前台投放数据）

* 生成素材质量报告（如 “高复用素材 TOP10”“低质量素材（需删除）TOP10”）

* 支持按素材类型、标签、复用率筛选统计结果

#### 操作权限



* 品牌专属管理员：仅可操作本品牌项目与素材

* 系统超级管理员：可操作所有项目与素材，查看存储全局统计

### 模块 5：投放数据看板（高阶功能，支撑前台步骤 9）

#### 功能描述

可视化展示各品牌脚本的投放效果数据，提供趋势分析与决策支持（MVP 版本暂不实现）。

#### 核心功能



1. 数据大屏：

* 全局数据：所有品牌脚本总播放量、总互动率（点赞 + 评论 + 收藏 + 分享）、总转化订单数、平均 ROI

* 品牌数据：按品牌展示核心数据（可切换单个品牌查看详情）

* 实时数据：展示近 24 小时 / 7 天数据趋势（折线图 / 柱状图）

1. 趋势分析：

* 行业爆款结构分析：基于第三方数据，展示本周 / 本月热门脚本结构 Top5

* 品牌效果趋势：展示单个品牌不同时期脚本的效果变化（播放量、转化率先）

* 卖点效果分析：统计不同卖点的转化效果，推荐高转化卖点

1. A/B 测试报告：

* 展示同一产品不同脚本的 A/B 测试结果（播放量、互动率、转化率对比）

* 自动标记胜出脚本，分析胜出原因（如结构公式、卖点植入方式）

* 支持导出 A/B 测试报告（PDF/Excel 格式）

#### 操作权限



* 品牌专属管理员：仅可查看本品牌数据看板与报告

* 系统超级管理员：可查看所有品牌数据，导出全局趋势分析报告

### 模块 6：系统与权限管理（基础支撑模块）

#### 功能描述

管理系统配置、用户账号、权限分配、操作日志，保障系统稳定安全运行。

#### 核心功能



1. 多租户隔离配置：

* 品牌租户管理：新增 / 编辑 / 禁用品牌租户（录入品牌名称、联系人、联系方式、开通时长）

* 数据隔离配置：确保各品牌数据（知识库、项目、素材、数据）独立存储，数据库层面隔离

* 品牌配置：为每个品牌配置专属域名、品牌 Logo、自定义皮肤（可选）

1. API 管理：

* 外部 API 配置：管理对接的 AI 模型 API（文生文、文生图、文生视频、TTS、数字人）的账号、Key、计费方式

* 第三方平台 API 配置：管理抖音 / 小红书 / 视频号 / 飞瓜等平台的 API 授权，支撑链接解析、数据回传、一键发布

1. 用户与权限管理：

* 用户账号管理：新增 / 编辑 / 禁用用户账号（系统超级管理员、品牌专属管理员、审核员）

* 角色权限配置：预设角色（超级管理员、品牌管理员、审核员），支持自定义角色并分配权限（按模块细粒度权限控制）

1. 操作日志与系统监控：

* 操作日志：记录所有用户的关键操作（账号登录、配置修改、审核操作、数据导出），包含操作人、操作时间、操作内容、IP 地址，不可删除

* 系统监控：监控服务器状态（CPU、内存、磁盘占用）、数据库状态、API 调用状态

* 告警配置：设置告警阈值（如服务器内存占用超 80%、API 调用失败率超 10%），配置告警方式（邮件 / 短信 / 企业微信）

仅系统超级管理员可操作
