# AI 智能体：爆款短视频脚本生成与复刻平台 - 前台（用户端）需求文档

## 文档信息



* **版本**：V1.0

* **状态**：脚本阶段 / MVP 至壁垒构建期

* **核心目标**：辅助品牌方运营 / 内容创作者，通过 AI 分析爆款结构，快速生成高转化、低风险的原创短视频脚本，完成从脚本到素材的全流程操作

* **使用人群**：品牌方运营、短视频内容创作者、营销人员

## 一、核心原则与风险提示

### 核心原则



1. 结构化操作：按步骤引导完成脚本生成，降低使用门槛

2. 合规优先：内置违禁词检测、原创度校验，规避内容风险

3. 灵活适配：支持爆款复刻与原创两种模式，满足不同创作需求

4. 数据闭环：素材可复用、版本可追溯、效果可追踪（高阶功能）

### 风险提示



* 数据源依赖：若爆款链接解析失败，可使用浏览器插件手动提取素材

* 知识产权：系统仅学习爆款 “结构特征”，不存储原文，生成脚本需用户确认原创度

* 内容实用性：AI 生成内容需人工审核调整，确保符合品牌调性与实际拍摄需求

## 二、完整工作流（9 步核心流程）

### 步骤 1：全局设定（基础参数配置）

#### 功能描述

用户设定本次视频制作的核心基础参数，为后续脚本生成提供方向指引。



| 字段   | 选项 / 输入方式                           | 说明                         | 默认值  |
| ---- | ----------------------------------- | -------------------------- | ---- |
| 视频比例 | 下拉选择：9:16（抖音竖版）/ 16:9（横版）/ 1:1      | 适配不同播放场景，影响分镜画面布局          | 9:16 |
| 视频类型 | 下拉选择：剧情口播 / 产品展示 / 教程干货 / 情感共鸣 / 其他 | 定义内容呈现形式，影响 AI 生成的文风与分镜逻辑  | 无    |
| 投放平台 | 下拉选择：抖音 / 小红书 / 视频号 / 快手            | 适配平台特性（如抖音重节奏、小红书重场景）与合规要求 | 无    |

#### 操作按钮

【下一步】→ 进入步骤 2：产品卖点输入

### 步骤 2：产品卖点输入（需求明确）

#### 功能描述

用户通过标准化模板填写产品核心信息，支持 AI 辅助优化，确保需求传递准确。

#### 标准化输入模板



| 字段     | 输入要求                      | 说明                   |
| ------ | ------------------------- | -------------------- |
| 产品名称   | 文本输入（必填）                  | 明确核心推广产品             |
| 产品特色卖点 | 文本输入（3-5 个，支持勾选主次）        | 区分核心卖点与辅助卖点，便于脚本重点植入 |
| 产品主卖点  | 单选（从特色卖点中选择，必填）           | 脚本核心突出的卖点，需明确唯一      |
| 产品辅助卖点 | 多选（从特色卖点中选择）              | 脚本中自然穿插，丰富内容层次       |
| 使用人群   | 文本 / 下拉选择（年龄、性别、场景、痛点，必填） | 明确目标用户，确保内容精准触达      |
| 其他要求   | 文本输入（可选）                  | 品牌调性、禁忌词、参考素材等补充说明   |

#### 核心功能按钮



* 【AI 优化 Brief】：调用大模型将用户口语化描述（如 “这个产品年轻人用着方便，能解决加班吃不上饭的问题”）整理为标准模板格式，提炼核心痛点与卖点

* 【下一步】→ 进入步骤 3：内容来源选择

* 【保存草稿】→ 将当前填写内容存入草稿箱，支持后续继续编辑

### 步骤 3：爆款链接 / 原创（内容来源选择）

#### 功能描述

用户选择脚本的内容来源模式，支持 “爆款复刻”（高效复用成功结构）或 “原创”（自主定义内容框架）。

##### 选项 A：爆款复刻（输入链接，推荐）



* 支持输入：抖音、小红书视频链接

* 系统自动处理：

1. 提取核心信息：完整文案 + 结构化数据（点赞 / 评论 / 收藏 / 分享 / 账号名称 / 视频描述）

2. AI 深度分析：输出「拉片式分镜报告」（拆解每个镜头的作用）+「结构公式」（如 “3 秒强痛点开头 + 场景化放大 + 产品方案 + 效果展示 + 限时优惠”）

* 用户操作：可手动修改 / 确认分析结果，确保结构符合预期

##### 选项 B：原创（自主创作）



* 跳过爆款分析环节，直接进入步骤 4

* 支持手动上传参考文案或自定义结构公式（如用户已有的成功脚本框架）

#### 操作按钮



* 【确认分析结果】（选项 A）/ 【进入原创脚本生成】（选项 B）→ 进入步骤 4

* 【重新输入链接】（选项 A）→ 更换爆款链接重新分析

### 步骤 4：分镜脚本生成与审核（核心环节）

#### 功能描述

AI 基于产品卖点 + 选定的结构公式（或原创框架）生成完整分镜脚本，支持在线编辑、合规校验与版本管理。

#### 生成的分镜脚本格式（表格展示）



| 镜号  | 景别  | 画面描述           | 台词 / 旁白             | 时长（秒） | 卖点植入说明        |
| --- | --- | -------------- | ------------------- | ----- | ------------- |
| 1   | 特写  | 主角皱眉看着加班餐，眼神疲惫 | “加班到深夜，想吃口热的怎么这么难？” | 3     | 铺垫痛点，为产品出场做准备 |
| ... | ... | ...            | ...                 | ...   | ...           |

#### 核心审核功能



1. 待审核区：AI 生成的脚本默认进入此区，支持在线编辑（增删分镜、修改画面描述 / 台词、调整时长）

2. 合规性检查：自动高亮《广告法》违禁词（如 “最”“第一”“国家级” 等），并给出合规替换建议

3. 原创度检测：与爆款分析提取的原文案库进行相似度比对，显示百分比（相似度＞60% 时弹窗提醒用户修改，避免侵权）

4. 版本管理：自动保存每次修改记录，支持查看历史版本并回退

#### 操作按钮



* 【提交审核】→ 脚本状态变为 “已审核”，进入步骤 5

* 【重新生成】→ AI 基于当前修改意见，重新生成完整分镜脚本

* 【保存草稿】→ 保留当前编辑进度，可后续继续处理

### 步骤 5：场景、角色、道具设定（视觉元素配置）

#### 功能描述

为审核通过的脚本，每个分镜配置视觉呈现元素，便于后续生成视频素材或指导拍摄。

#### 配置项



| 配置维度  | 操作方式                           | 说明                          |
| ----- | ------------------------------ | --------------------------- |
| 场景    | 下拉选择（室内 / 室外 / 办公 / 厨房等）或文本描述  | 明确镜头拍摄场景，影响画面风格             |
| 角色    | 下拉选择（角色库预设）或新建角色（填写年龄、性别、着装风格） | 定义视频中的人物形象，需符合目标人群定位        |
| 道具    | 文本输入或勾选已有道具（含产品本身、辅助展示道具）      | 明确每个镜头需要出现的物品，确保卖点展示清晰      |
| 风格参考图 | 图片上传                           | 支持上传参考图，为 AI 生图 / 生视频提供视觉指引 |

#### 操作按钮

【完成配置】→ 进入步骤 6：分镜视频生成

### 步骤 6：分镜视频生成（素材制作）

#### 功能描述

根据分镜脚本 + 视觉元素配置，生成对应镜号的视频片段，支持单独调整与标记。

#### 核心功能



* 生成方式：调用文生视频模型（如 Runway Gen-2、Pika 等 API），按分镜逐段生成

* 输出结果：每个分镜对应一个独立短视频片段（MP4 格式）

* 素材标记：支持对片段打标签（如 “可用”“产品特写好”“情绪到位”“需重制”），便于后续筛选复用

* 重新生成：支持单独选中某个片段，修改配置后重新生成

#### 操作按钮

【全部生成完成】→ 进入步骤 7：配音与对口型

### 步骤 7：配音与对口型（音频适配）

#### 功能描述

为生成的视频片段添加配音或实现对口型效果，提升视频完整性。

#### 两种模式选择



| 模式       | 适用场景          | 操作功能                                                                            |
| -------- | ------------- | ------------------------------------------------------------------------------- |
| TTS 旁白模式 | 无模特出镜、产品展示类视频 | 1. 选择音色库（支持多种男女声、方言、外语）2. 系统自动将台词 / 旁白转为语音 > 3. 支持调整语速、语调、音量 > 4. 支持手动上传自定义音频替换 |
| 对口型模式    | 有模特出镜、剧情类视频   | 1. 上传模特面部视频或选择数字人形象 2. 系统自动将台词与嘴型同步匹配3. 支持微调嘴型同步精度 > 4. 支持预设品牌专属数字人形象（需后台配置）    |

#### 操作按钮

【音频配置完成】→ 进入步骤 8：视频预览与素材管理

### 步骤 8：视频预览与素材管理（成品输出）

#### 功能描述

将所有分镜片段拼接为完整视频，提供编辑、预览、导出功能，同时管理所有素材。

#### 核心功能



1. 时间轴编辑：

* 自动拼接所有分镜片段（按镜号顺序）

* 支持调整片段顺序、裁剪时长、添加转场效果（如淡入淡出、滑动）

* 支持添加背景音乐（提供版权音乐库或上传自定义音乐）

1. 素材库管理：

* 所有生成的视频片段、配音、场景图、参考素材自动存入 “项目素材库”

* 支持按标签、镜号、卖点、生成时间等维度检索素材

* 支持素材收藏、删除、批量导出

1. 预览与导出：

* 支持 1080P/720P/480P 分辨率选择

* 预览完整视频，可逐帧查看细节

* 一键导出 MP4 文件，支持直接发布到绑定的抖音 / 视频号账号（需授权）

* 导出时自动生成视频封面建议（基于脚本核心镜头）

#### 操作按钮

【导出 / 发布完成】→ 进入步骤 9：投放数据追踪（高阶功能）

### 步骤 9：投放数据追踪（高阶功能）

#### 功能描述

追踪视频发布后的效果数据，为后续脚本优化提供依据（MVP 版本暂不实现，壁垒构建期迭代）。

#### 核心功能



* 监测链接生成：脚本生成时可选择自动植入优惠码 / 监测链接（如抖音购物车参数链接）

* 数据回传：通过 API 或第三方平台（飞瓜、蝉妈妈）获取播放量、点赞、评论、收藏、分享、转化订单数等数据

* A/B 测试：针对同一产品卖点，生成 3-5 种不同风格脚本，投放后自动对比各版本数据，展示效果排名

* 数据报表：可视化展示单视频 / 多视频的效果趋势，支持导出分析报告

## 三、MVP 版本最小功能集（优先实现）



1. 步骤 1：全局设定（核心参数配置）

2. 步骤 2：产品卖点输入（标准化模板 + AI 优化 Brief）

3. 步骤 3：爆款复刻（链接解析 + 结构公式输出）+ 原创选项

4. 步骤 4：分镜脚本生成与审核（分镜表格 + 违禁词高亮 + 原创度检测 + 版本管理）

5. 基础素材管理：保存生成的脚本与分析结果，支持导出
